# Задание №4-2

## Условие задачи

Создать тип, описывающий контакт в телефонной книге. Создать псевдоним типа телефонной книги (массив контактов) и реализовать для него интерфейс Sort{}.

## Запуск

```bash
go run main.go
```

## Тестирование

```bash
$ go run main.go
by number:
  1. Wtf 42
  2. Alex 123
  3. Bob 321

by name:
  1. Alex 123
  2. Bob 321
  3. Wtf 42
```
